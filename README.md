# NotesEngine

NotesEngine is a retrieval-augmented academic assistant designed to answer questions from study materials using contextual retrieval.

## Features (Planned)
- PDF ingestion with OCR fallback
- Semantic search using embeddings
- Answer generation using LLMs
- Telemetry logging
- Evaluation of response quality

## Architecture
Ingestion → Retrieval → Generation → Telemetry → Evaluation

## Status
Project setup initialized. Implementation in progress.
